# Getting started

A standalone RPC client and bundler for relaying UserOperations to the EntryPoint.

# Running an instance

See the `Client` documentation at [docs.stackup.sh](https://docs.stackup.sh/docs/category/client).

# Contributing

## Prerequisites

- Go 1.19 or later

## Setup

```bash
# Will install https://github.com/cosmtrek/air for live reloading
make setup-dev
```

## Run RPC server

```bash
make dev-rpc
```
